What is Staffing Intelligence

A real-time staffing management platform replacing spreadsheets with a live, AI-powered web app for professional services consulting teams.

  • Full-team availability across a rolling 12-week window at a glanceNo more chasing Excel files or asking around for availability
  • AI-powered staffing recommendations match open project needs to available consultantsClaude analyzes skills, level, availability, and utilization to suggest best fits
  • Real-time KPI dashboard — utilization, bench, overallocation, pipeline coverageOverview tab gives practice leadership an instant pulse on the team
  • Role-based access: admin, resource manager, project manager, executiveEach role sees exactly what they need — enforced at UI and API level
  • Ask Claude — natural language Q&A against live staffing data"Who's available for a 40h/wk R2R engagement starting in April?"

Features Built — V1 Stable + Phase 2 In Progress
Resource Allocation (Heatmap)
  • Rolling 12-week availability grid for all consultants
  • Always-on inline cell editing — click any cell to update hoursAirtable/Float model — no Edit Mode toggle
  • 4-tier color scheme: Red 0–10h / Yellow 11–44h / Green 45h / Orange 46h+

  • Global search — Ctrl+K / slash to find anyone instantly
  • Add project assignments directly from heatmap cells
Overview Dashboard
  • Available Capacity % — primary hours-based KPI
  • Utilization % — total booked / total available hours
  • On Bench — consultants with zero bookings this period
  • Overallocated Resources — anyone above 45h/wk with drilldown
  • Rolling Off Soon — upcoming availability alerts + Change Assignment CTA
  • All KPIs clickable — grouped by seniority with expand/collapse
AI Features
  • Open Needs pipeline — demand roles matched against supply
  • AI acceptance writes assignments to Supabase for full engagement date range
  • Ask Claude — natural language Q&A against live data
  • Skill + level + availability matching for recommendations
  • Rolling 90-day context window for accurate answers
Settings & Admin
  • User management — invite, activate, deactivate, change roles
  • Consultant profiles — skills, level, location, billability
  • Password strength enforcement for new invites
  • Role changes take effect in 30 seconds — no re-login needed
  • Soft deactivation — records preserved, fully reversible
Auth & Security

  • Row-Level Security on all database tables
  • Tab-level and API-level RBAC enforcement
  • Auto-logout on role change or deactivation
